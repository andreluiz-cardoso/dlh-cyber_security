# Linux Security Basics - 0x00

This directory contains scripts for Linux security tasks.

## Tasks
- 0-login.sh: displays last 5 login sessions
- 1-active-connections.sh: lists TCP connections with process info
- 2-incoming_connections.sh: allows TCP port 80
- 3-firewall_rules.sh: lists iptables rules
- 4-network_services.sh: shows listening TCP/UDP services
- 5-audit_system.sh: runs Lynis audit
- 6-capture_analyze.sh: captures 5 packets with tcpdump
- 7-scan.sh: scans live hosts with nmap
