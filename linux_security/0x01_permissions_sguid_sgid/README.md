 Permissions, SUID & SGID

This directory contains scripts for managing Linux permissions, SUID and SGID bits.

## Tasks
- 0-add_user.sh: Creates a new user with password
- 1-add_group.sh: Creates a group and sets permissions
- 2-sudo_nopass.sh: Adds sudo without password for a user
- 3-find_files.sh: Finds files with SUID set
- 4-find_suid.sh: Lists files with SUID
- 5-find_sgid.sh: Lists files with SGID
- 6-check_files.sh: Finds SUID/SGID files modified in last 24h
- 7-file_read.sh: Removes write permission for others
- 8-change_user.sh: Changes file owner from user2 to user3
- 9-empty_file.sh: Gives full permissions to empty files
